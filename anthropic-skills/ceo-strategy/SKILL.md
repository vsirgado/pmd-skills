## Framework 1 — OKR Planning

Use when: quarterly planning, annual goal-setting, "what should we focus on," prioritization conflicts.

**Structure:**
- 1 Company Objective (qualitative, inspiring, directional)
- 3 Key Results per Objective (quantitative, measurable, time-bound)
- Max 3 Objectives per quarter for a solo operator — constraint is real

**PMD OKR lens:**
1. Does this KR move a revenue metric directly?
2. Does this KR build leverage (systems, content, audience) that compounds?
3. Is this measurable in 13 weeks?

If no to all three → it's not a KR, it's a task.

**Quarterly Review Protocol:**
- Score each KR 0.0–1.0 (0.7 = success; 1.0 = set too easy)
- What's the root cause of any KR below 0.5?
- What carries forward vs. gets dropped?
- What did we learn that changes next quarter?

## Framework 2 — Business Model Design

Use when: designing new revenue streams, stress-testing existing ones, productizing services, pricing architecture.

**Business Model Canvas (PMD version):**
Apply across each of Victor's 4 revenue streams:

| Element | Questions to answer |
|---------|-------------------|
| Value Proposition | What specific outcome do we deliver? For whom? Why us? |
| Customer Segment | Who pays? Who decides? Who benefits? |
| Revenue Model | One-time? Recurring? Usage? Outcome-based? |
| Key Activities | What must we do exceptionally well? |
| Unfair Advantage | What do we have that's hard to copy? |
| Cost Structure | What are the real costs? Time, money, attention? |

**Productization test** — for any service being considered as a product:
1. Can the output be standardized without losing quality?
2. Can delivery be systemized or AI-automated?
3. Is there a market willing to pay without a custom sales process?
4. What's the minimum viable version?

## Framework 3 — Competitive Strategy

Use when: positioning decisions, entering new markets, defending against competition, evaluating market opportunity.

**Porter's 5 Forces (PMD application):**
- Competitive rivalry: other healthcare marketing agencies + AI tools
- Threat of new entrants: any marketer with GHL access
- Buyer power: healthcare practices have options; loyalty = retention moat
- Supplier power: GHL, AI APIs — manageable but watch dependency
- Substitutes: in-house hiring, other AI tools doing PMD's job

**Moat analysis — rate each 1–5:**
- Switching costs (how painful to leave PMD?)
- Network effects (does each client make PMD more valuable to others?)
- Proprietary data/systems (PMD's GHL workflows, content systems, AI stack)
- Brand and authority (Victor's expert positioning in healthcare AI)
- Scale advantages (does size help PMD win? Mostly no — lean is the moat)

**Blue Ocean move** — when evaluating new positioning:
- What does every player in the space compete on? (eliminate or reduce)
- What does no player currently offer? (create or raise)
- Result: a strategy canvas that shows PMD's unique value curve

## Framework 4 — Strategic Decisions

Use when: build vs. buy vs. partner, major pivots, resource allocation, opportunity evaluation.

**Build / Buy / Partner / Kill matrix:**

| Option | When to choose |
|--------|---------------|
| Build | Core to moat, high leverage, can be systematized |
| Buy | Faster than building, cost < build cost + time |
| Partner | Complements PMD without diluting focus |
| Kill | Doesn't meet revenue-first test; drains attention |

**Opportunity scoring** (rate each 1–5):
1. Revenue impact in 90 days
2. Revenue impact in 12 months
3. Leverage created (does this make future work easier?)
4. Alignment with PMD's moat
5. Effort required (inverse — less effort = higher score)

Total score ÷ 5 = priority ranking. Anything below 3.0 → deprioritize.

## Framework 5 — Horizon Planning

Use when: annual planning, long-term vision, deciding what to build next.

**Three Horizons:**
- **Horizon 1 (now–90 days):** Optimize what's working. Client delivery, course MVP, GHL operations.
- **Horizon 2 (90 days–12 months):** Build the next engine. Course flywheel, SaaS productization, audience growth.
- **Horizon 3 (12 months–3 years):** Define the category. PMD as the AI marketing OS for healthcare. The brand that trains the industry.

Rule: Horizon 1 funds Horizon 2. Horizon 2 funds Horizon 3. Never sacrifice H1 stability for H3 ambition.

**Strategic questions to ask annually:**
1. What business are we actually in? (Is it still the right one?)
2. What would have to be true for us to 10x in 3 years?
3. What's the one constraint that, if removed, changes everything?
4. What are we doing that no one else can do?
5. What should we stop doing entirely?

## Output Format

**For OKR planning:** Present Objective + 3 KRs with scoring criteria. One table per objective.

**For strategic analysis:** Problem → Framework applied → Insight → Recommendation → First action.

**For opportunity evaluation:** Score each option on the 5-point matrix. Rank. Recommend top choice with one-line rationale.

**For horizon planning:** Three-column view (H1 / H2 / H3) with concrete milestones per horizon.

Always end with: **Strategic priority** — the one thing that matters most right now given everything above.
